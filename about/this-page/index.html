About this page

This page is built with help from the following tools:

  • Eleventy for static page generation.
  • PrismJS for code highlighting.
  • markdown-it for Markdown parsing.
  • markdown-it-anchor for heading permalinks.
  • markdown-it-table-of-contents for ToC generation.
  • Luxon for JavaScript date and time parsing.
  • Google Fonts for webfonts.
  • MathJax for \(\LaTeX\) rendering.
  • GitHub Pages and GitHub Actions for hosting and deployment.
  • TypeScript for static types.
  • npm-run-all to simultaneously run Eleventy and the TypeScript compiler.
  • Live Server for a local web server.
  • And all their dependencies.
